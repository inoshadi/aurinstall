# aurinstall.sh
Simple Arch User Repository (AUR) package installation

- set as executable
```
chmod a+x aurinstall.sh
sudo cp -p aurinstall.sh /usr/bin/
```

- example
```
aurinstall.sh some-aur-pkg
```
